# 54. 螺旋矩阵（中等）
## 题目：
给你一个 `m` 行 `n` 列的矩阵 `matrix` ，请按照 **顺时针螺旋顺序** ，返回矩阵中的所有元素。
## 题解：
按照一圈一圈去模拟，每一圈的初始位置很好确定，之后按照逻辑写遍历。最后一个上升的竖边取决于前一个横边存不存在，判断一下。
```c++
class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        vector<int> res;
        int m=matrix.size();
        int n=matrix[0].size();
        int cnt=min(ceil((double)m/2),ceil((double)n/2));
        for(int i=0;i<cnt;i++){
            int row=i;
            int col=i;
            bool flag1=false;
            while(col<n-i){
                res.emplace_back(matrix[row][col]);
                col++;
            }
            col--;
            row++;
            while(row<m-i){
                res.emplace_back(matrix[row][col]);
                row++;
            }
            row--;
            col--;
            while(col>=i&&row>i){
                res.emplace_back(matrix[row][col]);
                col--;
                flag1=true;
            }
            col++;
            row--;
            while(row>i&&flag1){
                res.emplace_back(matrix[row][col]);
                row--;
            }
        }
        return res;
    }
};
```
